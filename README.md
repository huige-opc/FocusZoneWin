# FocusZoneWin

中文 | [English](README.en.md)

全屏专注区遮罩工具。录制课程、直播演示时，屏蔽屏幕非核心区域，让观众视线聚焦。

![FocusZoneWin](docs/cover.png)

## 效果

![开启遮罩后只有框内保持原样，其余全部变暗](docs/effect.png)

全屏暗幕覆盖桌面，只在选区范围内挖出亮区，类似"聚光灯"效果。控制条可浮动在任何位置。

## 下载

从 [Releases](https://github.com/huige-opc/FocusZoneWin/releases) 下载最新版，解压后双击 `FocusZoneWin.exe` 即可运行。

> ⚠️ 首次运行可能提示"Windows 已保护你的电脑"，点击"更多信息" → "仍要运行"即可。这是因为本工具没有购买代码签名证书，不影响使用。你也可以在 Releases 页面查看 VirusTotal 检测结果确认安全。

![控制条：选区 / 遮罩 / 暗度 / 颜色 / 线型 / 录屏隐藏 / 退出](docs/toolbar.png)

| 操作 | 方式 |
|------|------|
| 进入选区 | 点击控制条 **▶ 选区** 或 `Ctrl+Shift+S` / 双击托盘图标 |
| 框选区域 | 进入选区后画面冻结，鼠标拖拽画框，松开锁定 |
| 确认退出 | 按 `ESC` 或鼠标右键 |
| 调节暗度 | 拖动 **🌙 滑条**（0~100%） |
| 切换边框颜色 | 点击 **■ 颜色按钮**（蓝/绿/橙/红/紫/青） |
| 切换线型 | 点击 **⬜ 虚线/实线** |
| 录屏隐藏 | 点击 **🔴 录屏** 切换控制条对 OBS/投屏可见性 |
| 退出程序 | 点击 **✕ 退出** |
| 移动控制条 | 鼠标左键按住拖动 |

## 选区流程

1. 点击 **▶ 选区**
2. 画面冻结（截图定格），颜色和暗度滑条自动冻结
3. 鼠标拖拽框选专注区域，松开锁定
4. 按 **ESC** 或 **右键** 确认退出，暗幕生效
5. 再次按 **ESC** 或 **右键** 关闭暗幕，控制条恢复可调

> 选区期间画面保持冻结，直到确认退出。可以在同一轮选区中反复调参数再框选。
